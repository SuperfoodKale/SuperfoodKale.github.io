# Projects

This folder contains software and technical systems that have been built independently.

## What is included
Each project documents:
- Purpose and overview
- System design and architecture
- Key technical decisions
- Challenges and solutions
- Link to external source code repository

## Purpose
Projects demonstrate ability to:
- Design and build functional systems
- Work with software engineering concepts
- Develop tools and automation outside of guided tasks

## Structure
Information about each project is stored in its own folder:

example:
- /file-parser/
- /media-cas-engine/

Each folder contains a README.md describing the project and linking to its external repository.

## Goal
To show practical engineering ability and independent system design thinking.
