# Assets

This folder contains supporting media used across the portfolio site.

## Contents
- Images used in case studies and project writeups
- Diagrams explaining system design or troubleshooting flow
- Screenshots used as evidence in technical investigations

## Purpose
Assets here are not standalone content. They are referenced by:
- Case studies
- Project documentation

## Notes
- Keep files organised in subfolders such as `/images` and `/diagrams`
- Prefer clear filenames that match the related case study or project
